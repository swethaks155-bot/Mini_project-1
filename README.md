This project focuses on analyzing a synthetic online retail sales dataset to gain meaningful insights into customer behavior, product performance, and overall sales trends. The dataset simulates real-world e-commerce transactions, making it ideal for practicing data analytics and visualization techniques.

The project begins with data cleaning and preprocessing in Microsoft Excel, where missing values, inconsistencies, and data formatting issues are handled to ensure accuracy. The cleaned dataset is then imported into Power BI, where an interactive dashboard is created using various visualizations such as KPI cards, bar charts, line graphs, maps, and pie charts.

The dashboard provides insights into key business metrics including total revenue, total orders, product-wise performance, monthly sales trends, and payment methods. This project demonstrates how raw data can be transformed into actionable insights, helping businesses make informed and data-driven decisions.# Mini_project-1
